# Project 3 — SQL Case Study: Logistics & Delivery Performance (Olist)

## What this case study answers
- What is the **on-time delivery rate**, and how does it change over time?
- Which **states** have the best/worst on-time performance?
- How do delivery days and freight cost vary by geography?
- Does late delivery impact customer experience (review score)?
- Which states/lanes are **high-risk** due to high volume + low on-time rate?

## How to run (SQLite)
1) Create a database in **DB Browser for SQLite** (or any SQL tool).
2) Run `schema.sql` to create the `deliveries` table.
3) Import the CSV `data/olist_delivery_kpis_sample_20000.csv` into the `deliveries` table.
4) Run the queries inside `queries.sql`.
